# فيت — موقع MVP

موقع عرض لتطبيق فيت الصحي: مدرب شخصي مصوّر، جدول غذائي من أخصائي، تتبع النشاط والسعرات، وقسم الإصابات.

## تشغيل الموقع

- **من المتصفح مباشرة:** افتح الملف `index.html` في أي متصفح.
- **سيرفر محلي (اختياري):**
  ```bash
  npx serve .
  ```
  ثم افتح الرابط الذي يظهر (مثلاً `http://localhost:3000`).

## المحتوى الحالي

1. **المدرب الشخصي:** جدول يومي، فيديو تمارين، تواصل مع المدرب.
2. **التغذية:** جدول غذائي من أخصائي، مخصص لأهدافك.
3. **التتبع:** سعرات، نشاط، تقارير وتقدم.
4. **الإصابات:** تسجيل إصابات، تمارين بديلة، تواصل مع الفريق.
5. **قائمة انتظار:** نموذج بريد للإطلاق والعروض.

## ملاحظات

- النموذج حالياً يعرض رسالة تأكيد فقط (لا يرسل البيانات لسيرفر). لربطه بقاعدة أو خدمة بريد لاحقاً تحتاج backend أو خدمة مثل Formspree / ConvertKit.
- الخط: IBM Plex Sans Arabic (تحميل من Google Fonts).
