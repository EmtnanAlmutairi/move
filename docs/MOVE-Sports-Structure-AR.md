# 🧠 الهيكل النهائي والمطور للرياضات داخل MOVE

## Strength & Muscle (بناء العضلات والقوة)
### Subcategories
- Bodybuilding
- Strength Training (Powerlifting style)
- Home Strength (بدون نادي)

### Tracking Metrics
- الوزن المستخدم (kg)
- عدد التكرارات (Reps)
- عدد الجولات (Sets)
- 1RM (أقصى وزن)
- حجم التمرين (Volume)
- صور التقدم
- قياسات الجسم (صدر / ذراع / خصر)

### User Journey
- يختار هدف (زيادة عضل / تضخيم / إعادة بناء)
- ياخذ برنامج أسبوعي
- يسجل كل تمرين
- التطبيق يحسب التقدم تلقائي

### تطوير إضافي
- إضافة Tempo و RPE (معدل الجهد المحسوس)
- تنبيهات زيادة الحمل الدوري (Progressive Overload Alerts)
- مثال: "حان وقت زيادة الوزن بمقدار 2.5kg"

### Tracking Metrics

- السعرات المحروقة

- يشوف تغيره أسبوعيًا
